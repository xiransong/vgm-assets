# vgm-assets Docs

This directory keeps longer-lived design notes, working plans, and catalog
notes for `vgm-assets`.

## Structure

- `architecture/`
  - repo boundaries, storage layout, source policy, export rules, and current
    consumer contracts
- `catalogs/`
  - notes tied to specific asset or material catalog slices
- `plans/`
  - active short-term plans for the next implementation steps
